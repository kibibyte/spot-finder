# View spot finder

## Build

```shell script
$ ./gradlew build
```

## Test

```shell script
$ ./gradlew test
```

## Run

```shell script
$ java -jar build/libs/view_spot_finder-1.0-SNAPSHOT-all.jar <mesh file> <number of view spots>
```
